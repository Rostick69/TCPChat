# Многопользовательский TCP-чат

Лабораторная работа №4 по дисциплине «Системное программное обеспечение»  
Многопользовательское приложение для обмена сообщениями в локальной сети 🚀

## Что умеет приложение

- Подключение нескольких клиентов одновременно — каждый в отдельном потоке
- Рассылка сообщений всем участникам чата (broadcast)
- Уведомления о входе и выходе участников
- Список пользователей онлайн в реальном времени
- Корректная обработка внезапного отключения клиента
- Тёмный интерфейс на WPF для сервера и клиента

## Как запустить

1. Открой решение `TCPChat.sln` в Visual Studio
2. Правой кнопкой на решение → **Properties** → **Multiple startup projects** → для Server и Client поставь **Start**
3. Нажми **F5** — откроются оба окна
4. В окне сервера нажми **Запустить**
5. В окне клиента введи никнейм и нажми **Подключиться**
6. Для второго клиента запусти ещё один экземпляр: правой кнопкой на **Client** → **Debug** → **Start New Instance**

## Протокол

| Префикс | Формат | Описание |
|---------|--------|----------|
| `/join` | `/join Никнейм` | Подключение к серверу |
| `SYS:` | `SYS:текст` | Системное уведомление |
| `MSG:` | `MSG:ник:текст` | Обычное сообщение |
| `USERS:` | `USERS:ник1,ник2` | Список пользователей онлайн |
